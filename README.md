# ⚡ Instalação

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/RCruzm08/Nexus.git
```

## 2️⃣ Entre na pasta

```bash
cd Nexus
```

## 3️⃣ Execute em um servidor local

Você pode utilizar:

- XAMPP
- Laragon
- WAMP
- PHP Built-in Server

---

# ▶️ Rodando com PHP local

```bash
php -S localhost:8000
```

Depois abra:

```bash
http://localhost:8000
```
